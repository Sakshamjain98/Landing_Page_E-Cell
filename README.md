# Particle.js and Lottie Landing Page

## [VISIT](https://sakshamjain98.github.io/Landing_Page_E-Cell/)

## Description

This landing page project is created using HTML, CSS, JavaScript, Particle.js, and Lottie animation library.
## Features

- **Particle Effects:** Utilizes Particle.js library to create dynamic and animated particle effects in the background.
- **Lottie Animation:** Incorporates Lottie animation from e-cell-kier to add visually appealing and interactive elements to the landing page.

## Technologies Used

- HTML
- CSS
- JavaScript
- Particle.js

## Usage

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
